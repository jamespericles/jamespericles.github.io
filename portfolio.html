Password Generator

This simple project is a Javascript powered HTML password generator. It allows users to select from several different variables to generate their password with, then displays the password on screen. It also includes a button to allow users to copy the generated password to their clipboard.

Password Generator iOS App

Through video tutorials, I was able to teach myself Swift during a slower weekend of my Boot-Camp. For my first project, I decided to take my Password Generator project and turn it into an iOS app!

Informed Diner

This was an exciting opportunity to work with several classmates to create an app of our own. Utilizing several Google Places API calls and an API called BestTime, we were able to create an app that gives the user the ability to search for restaurants and provide for them a safety score based on how busy the location is. BestTime utilizes Google's phone traffic to calculate the foot traffic at different locations.

JQuery Work Day Planner

This simple planner integrates both JQuery and BootStrap technologies to create a simple 9 to 5 event planner. The user is able to add events to each time block. The added events are then saved to localstorage so that the user can leave the page without their added events disappearing. Each time block changes color based on the time of day.
